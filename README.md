# 生活必需品かんばんアプリ

トヨタ生産方式（TPS）の**かんばん方式**を、日常の在庫管理に適用したWebアプリ。

## 概要

- 日用品（ティッシュ、洗剤などの消耗品）を「かんばん方式」で管理するWebアプリです。
- 家にある在庫の最後の1つを開封したら、かんばんを発行して買い物リストに表示します。
- 購入後は買い物リストから在庫リストへ戻し、日用品の補充をシンプルに管理できます。

## コンセプト

> 最後の1つを開封したら「かんばん発行」→ 買い物リストに自動反映 → 買ったら在庫に戻る。
> 

工場の部品補充と同じ仕組みを、日用品の買い物に応用する。

「無くなりそう」に気づいた瞬間にタップするだけで、買い忘れを減らせるようにした。

## 作った背景

- 日用品の買い物に行く回数を減らしたかった。
- 在庫管理をもっと簡単にしたかった。毎回メモに書くのではなく、「無くなったらタップ」で管理できる形にしたかった。
- 最終購入日を記録し、日用品の使用ペースを把握したかった。

## 主な機能

- **在庫ボード** — 全アイテムをカード表示し、タップで「かんばん発行」
- **買い物リスト** — 発行中のアイテムだけを表示し、タップで「購入完了」→ 在庫に戻す
- **アイテム管理** — 追加・削除・編集に対応
- **カテゴリ分け** — 食品 / 日用品などで整理
- **店舗ごとのグルーピング** — どの店で買うかを見える化
- **日時記録** — かんばん発行日・購入日を自動で記録
- **データ永続化** — localStorageで状態を保持


## セットアップ

1. リポジトリをクローン
2. `npm install`
3. `npm start`

## 使い方

1. 「管理」タブで生活必需品アイテムを追加する
2. カテゴリと店舗を設定し、在庫ボードに反映する
3. 在庫が無くなったら在庫ボードのカードをタップし、かんばんを発行する
4. 買い物リストで購入したカードをタップし、在庫ボードへ戻す
5. かんばん発行日と最終購入日を記録して、補充タイミングを振り返る
